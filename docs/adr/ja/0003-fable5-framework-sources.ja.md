# 0003: Fable 5 の2つのフレームワークソースを取り込む(公式プロンプティングガイド + リーク版システムプロンプト)

(English: [0003-fable5-framework-sources.md](../en/0003-fable5-framework-sources.md))

## Context(背景)

ユーザーから、Claude Fable 5 の思考フレームワークを記述した2つのドキュメントをプラグインに取り込むよう依頼された(必要なら別スキルへの分離も可)。

1. Anthropic 公式ガイド *Prompting Claude Fable 5*(`platform.claude.com/docs/en/build-with-claude/prompt-engineering/prompting-claude-fable-5`)
2. `elder-plinius/CL4R1T4S` のリーク版 Fable 5 本番システムプロンプト(`ANTHROPIC/CLAUDE-FABLE-5.md`)

2つのソースは性質が異なる。公式ガイドは (a) Fable 5 が作業中に従う行動規則と、(b) Fable 5 を「使う側」向けのガイダンス(effort調整、スキャフォールディング、ツール設計、移行)が混在している。リーク版プロンプトは大半がコンシューマーアプリ固有の設定(ファイルパス、画像検索、著作権制限)で、転用可能な核(トーン、フォーマット最小主義、ミス対応)は小さい。

## Decision(決定)

発火スコープで分割する(ADR 0002 と同じ理屈):

- **行動規則 → `skills/fable-coding/SKILL.md` に統合**(v1.4.0): 情報が揃ったら動く/過剰計画の抑制(§0)、ツール結果と照合した進捗報告(§5)、最終サマリの可読性+散文優先+ミスを率直に認める(§7、後者2つはリーク版由来)、assess と fix の境界+状態変更コマンド前の証拠確認+約束でターンを終えない(§9)、アンチパターン4件追加。これらはコーディングタスクで発火すべき内容であり、既存スキルの発火条件と一致する。
- **使う側向けガイダンス → 新スキル `skills/prompting-fable-5/SKILL.md`**: effort レベル、Anthropic の検証済みスニペットを原文のまま収録、サブエージェント/メモリ/send_to_user のスキャフォールディング、Opus 4.8 への refusal フォールバック、`reasoning_extraction` の落とし穴、Opus→Fable 移行チェックリスト、リーク版から蒸留した「デフォルトペルソナ」節。Fable 5 向けのプロンプト・エージェント・スキルを書くときに発火する — コーディングスキルが乗っ取ってはいけないスコープ。
- リーク版のコンシューマーアプリ向け内容(著作権の語数制限、画像検索ルール、アップロードパス、製品カタログ)は意図的に**不採用**: claude.ai アプリの設定であり、コーディング/思考フレームワークではない。

## Alternatives rejected(却下した代替案)

- **単一の「fable-thinking」統合スキル** — 粒度が誤り: 半分は全コーディングタスクで発火すべき内容、残り半分は Fable 5 上に構築するときだけ必要。1つの description では両方に正しく発火できない。
- **公式ガイドをコーディングスキルへ丸ごとコピー** — 大半(タイムアウト、send_to_user の JSON、API パラメータ)はコーディングタスク中には無関係で、規律スキルを薄める。
- **リーク版システムプロンプトの全文取り込み** — 大部分はアプリの配管であり、転用可能な行動規則の核だけが収録に値する。

## Consequences(帰結)

- Fable 5 向けのプロンプトエンジニアリング作業で、記憶からの言い換えではなく Anthropic の検証済み文言が原文のまま使えるようになる。
- コーディングスキルが公式文書化された「捏造防止」「境界」規則を獲得し、エミュレーションが Anthropic 自身のステアリング指針と整合する。
- 各英語スキルには人間用の参考訳 `SKILL.ja.md` を併設。Claude が読み込むのは `SKILL.md` のみのため、編集は英語版が先、和訳が追随する。
- 天井: 両ソースはスナップショット(ガイドは 2026-07-04 取得; リーク版はコミュニティによる書き起こしで未検証)。リーク版由来の内容は本 ADR で出所を明示。Anthropic がガイドを改訂した場合、`prompting-fable-5` は手動更新が必要。
